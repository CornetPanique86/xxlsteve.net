    FAQ

    Frequently Asked Questions

    What video editor do you use? I use Davinci Resolve 17 (the free version), which I highly recommend!
    What do you use to make addons? / How do I make an addon?

    I primarly learnt how to make addons using wiki.bedrock.dev, which is a community-driven website with tons of super helpful tutorials related to MCBE addons! There also is bedrock.dev for resources and info while developping addons.

     

    I use the Visual Studio Code text editor along with the Blockception Minecraft Bedrock Development, Snowstorm Particle Editor and UUID Generator extensions. bridge. is also a very cool editor which is made for MCBE addons, but I’m personally more used to just using VSCode.

     

    For models, I use Blockbench.

    For image editing, I use paint.net.
